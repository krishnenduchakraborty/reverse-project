# reverse-project
🔍 Reverse Project

git clone https://github.com
cd reverse-project


This is my personal playground for reverse engineering and low-level tinkering. It’s basically a collection of everything I’ve learned while trying to take things apart, see how they tick, and (occasionally) put them back together in ways they weren't intended to work.

What I’m working on here: Messing with Binaries: Learning how to patch executables and swap out logic directly in the hex.

Code-to-Machine: Writing small programs in C and C++ just to disassemble them and see what the compiler actually does to my code.

Automation: I got tired of doing the same tasks twice, so there are some Python and Shell scripts in here to handle the boring stuff.

🧠 Why this exists I've always been curious about what’s happening "under the hood." This repo is where I document my progress, keep my helper scripts, and save my "aha!" moments while digging through assembly and hex dumps.
